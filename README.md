## Waves Music App
Music app created with React.js

Contains:
* Responsive Design
* Sass for styling
* Fontawesome 
* React Hooks for simple state management
* Library UI 
* Functional music player

# Project Setup
```
clone this repository

run npm install

run npm start
```
